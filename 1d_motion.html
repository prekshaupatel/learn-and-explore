Notes

To solve for variables in 1-dimensional motion, make sure that the signs for initial velocity (u), final velocity (v), acceleration (a), and displacement (d) are correct.

For example, let us think of a problem set along the y-axis (up is positive, down is negative).

  • Velocity
    • If you initially throw a ball upwards, the initial velocity will be positive.
    • If you throw a ball downwards, the initial velocity will be negative.
  • Acceleration
    • Since gravity pulls everything downwards, the acceleration due to gravity is negative.
  • Displacement
    • If an object ends up higher than where it initially was, the displacement is positive.
    • If an object ends up lower than where it initially was, the displacement is negative.
